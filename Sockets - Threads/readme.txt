

------------
Περιληπτικά:
------------



Ο φάκελος SViktoras-Prj3 περιέχει το ContentServer.c,το MirrorServer.c,το MirrorInitiator.c και το makefile.
Πιο αναλυτικά:



- ContentServer.c : Πρόκειται για το πρόγραμμα που εκτελείται από απομακρυσμένες συσκευές και εξυπηρετεί
 
τα αιτήματα LIST και FETCH.


- MirrorServer.c  : Το πρόγραμμα MirrorServer.c εκτελείται από μια συσκευή η οποία παριστάνει τον ρόλο 
του
 server.Δέχεται αιτήματα από τον Mirror Initiator και στη συνέχεια επικοινωνεί με τις απομακρυσμένες
συσκευές προκειμένου να αντιγράψει τοπικά οποιοδήποτε αρχείο. 

- MirrorInitiator.c : Πρόκειται για το πρόγραμμα που εκτελείται από μια συσκευή η οποία παριστάνει τον 
ρόλο του client.Επικοινωνεί με τον Mirror Server και τον ενημερώνει με ποιες συσκευές θα συνδεθεί 
προκειμένου να αντιγράψει αρχεία.



---------
Εκτέλεση:
---------



Αρχικά εκτελούνται οι Content Servers,οι οποίοι παραμένουν σε λειτουργία καθ΄όλη τη διάρκεια εκτέλεσης 

του Mirror Server.
Μια ενδεικτική εκτέλεση του προγράμματος από την γραμμή εντολών είναι η ακόλουθη:



./ContentServer -p 9007


Στη συνέχεια εκτελείται ο Mirror Server,ο οποίος αναμένει εντολή από τον Mirror Initiator.
Μια ενδεικτική 
εκτέλεση του προγράμματος από την γραμμή εντολών είναι η ακόλουθη:



./MirrorServer -p 8000 -m FILES



Τέλος εκτελείται o Mirror Initiator ο οποίος στέλνει τις πληροφορίες για τη σύνδεση του Mirror Server 

με τις απομακρυσμένες συσκευές.
Μια ενδεικτική εκτέλεση του προγράμματος από την γραμμή εντολών 
είναι η ακόλουθη:



./MirrorInitiator -n linux28.di.uoa.gr -p 8000 -s linux23.di.uoa.gr:9007:Dir1:5,linux29.di.uoa.gr:9008:Dir2:8
