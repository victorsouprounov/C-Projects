5 αυτόνομα προγράμματα (circle.c,semicircle.c,ring.c,square.c,ellipse.c).
Καθένα από αυτά μπορεί να εκτελεσθεί αυτόνομα και να παράγει το αποτέλεσμα,σύμφωνα με τα ορίσματα που θα δώσει
σαν είσοδο ο χρήστης.Καθένα από αυτά,αφού τερματίσει,δημιουργεί ένα ASCII αρχείο το οποίο περιέχει τα αποτελέσματα.
Επιπλέον,στον φάκελο υπάρχει και το shapes.c.Μια ενδεικτική εκτέλεση του προγράμματος shapes είναι η ακόλουθη:

./shapes -i test6 -w 2 -d Temp

Στη συνέχεια ο χρήστης μέσω ενός cli δίνει τα utilities τα οποία επιθυμεί να εκτελέσει με τα κατάλληλα ορίσματα.
Μια ενδεικτική γραμμή εντολής στο cli είναι η ακόλουθη:

circle 1.0 1.0 10.0, ellipse 3.0 3.0 2.0 3.0, ring 5.0 5.0 4.0 4.5;

Σε αυτή την περίπτωση το πρόγραμμα shapes θα δημιουργήσει 3 handlers και workers όσους έχουν δοθεί σαν όρισμα στην
γραμμή εκτέλεσης του προγράμματος shapes.Οι handlers με τη σειρά τους θα δημιουργήσουν τους workers οι οποίοι είναι 
υπεύθυνοι για την εκτέλεση των utilities.Κάθε worker που εκτελεί το utility,δημιουργεί ένα fifo file το οποίο περιέχει
τα αποτελέσματα από την εκτέλεση του utility.Αφού κάθε handler ολοκληρώσει την δουλειά του,στην οθόνη εκτυπώνεται το
process id του καθώς και το status με το οποίο τερμάτισε.
